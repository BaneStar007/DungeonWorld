How to use this guide

Below is a list of all 770+ symbols included in Hermetica, which will make it very easy to select a symbol to use with either the Hermetica webfont on a web page or the desktop font. For the best viewing, you'll want to install the Hermetica desktop font on your computer first.

To use a symbol with the desktop font, you can either use your computer's character map or your application's symbol picker (Hermetica symbols occupy the Unicode Private Use Range), or you can copy the symbol from this guide by clicking the small grey instance of any symbol in the lower right (beside the unicode value, e.g. U+EBEB). If you don't have the desktop font installed on your system, it's likely the symbol will appear as either an empty rectangle or invisible, but you can still click to copy it, and when you paste it into the program of your choice and choose Hermetica as the font, you'll see your symbol!

To use Hermetica as a webfont, you'll first need to install the font on your web page by including the Hermetica CSS file and webfont files on your website. Then all you need to do to use a symbol is to create an empty <span> tag with the correct class name of the symbol you want. Simply click on the class name below each symbol in this guide to copy it! (Note: the included CSS file includes ALL 770+ symbols - if you only need to use a few, you can save space (and loading time) by only including the CSS class definitions for the symbols you want.)
