Welcome to the Instructions

The Following Pages contain "easy to create" PC & NPCs for Dungeon Delvers Twelve

The Links on the Left have pre-build characters for the Main twelve Archtypes, with a drop down for characters of 1st, 4th, 8th and 12th ranks. Lastly, a Work in Progress system for making your own characters from scratch.

Each Section of Character Creation is able to be swapped out with Dungeonworld Character Creation, if desired.

The Default Structure is:

<header>
First Page Header Text
</header>

<section> // Sections are repeated
<subsection> // Sections are repeated, but define if the next part is a row, grid or box
IF (subsection is a row) {
<form-row></form-row>

}
IF (subsection is a grid) {
<form-grid></form-grid>

}
IF (subsection is a component) {
<form-component></form-component>

}
</subsection>
</section>

So, This:

{ sections:
[{
sectionTitle: "Section #1",
subsections:
[{
type: row,
title: "welcome"
},
{
type: row,
title: "To the Jungle"
}]
}]
}

Becomes This:

Section #1
Welcome
To the Jungle
